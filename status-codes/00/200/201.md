# HTTP Status Code - 201

Client receives 201 when server successfully creates inner object
HTTP Status Code: <span style="color:green">201</span>
```json
"success": {
    "code": "002010",
    "attributes": { }
}
```

or if attribute provided by server is present

HTTP Status Code: <span style="color:green">201</span>
```json
"success": {
    "code": "002011",
    "attributes": { 
        "message": "string"
    }
}
```
