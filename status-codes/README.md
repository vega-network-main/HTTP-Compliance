# Global codes
This folder aims to get general HTTP status codes specs and JSON-s

Example:
HTTP Status Code: <span style="color:green">200</span>
```json
"error": {
    "code": "002000",
    "attributes": {
        "attr1":  "attr1",
        "attr2":  "attr2",
    }
}
```
